https://leetcode.cn/problems/maximum-element-after-decreasing-and-rearranging/description/?envType=daily-question&envId=2026-06-28



排序后,后面每个都尽量+1

```java
    class Solution {
        public int maximumElementAfterDecrementingAndRearranging(int[] arr) {
            Arrays.sort(arr);
            arr[0] = 1;
            for (int i = 1; i < arr.length; i++) {
                arr[i] = Math.min(arr[i - 1] + 1, arr[i]);
            }
            return arr[arr.length - 1];
        }
    }
```



官解 O(n)

因为a[0]=1,每个差值小于等于1,所以最优情况就是1,2,3,4...n, 最大就是n

```java
class Solution {
    public int maximumElementAfterDecrementingAndRearranging(int[] arr) {
        int n = arr.length;
        int[] cnt = new int[n + 1];
        for (int v : arr) {
          // 大于n的当做n
            ++cnt[Math.min(v, n)];
        }
        int miss = 0;
        for (int i = 1; i <= n; ++i) {
            if (cnt[i] == 0) {
              // 如果没有,说明需要有更大的数变小来进行补充
                ++miss;
            } else {
              // 如果有多余的,那么可以补充给前面缺失的小的
                miss -= Math.min(cnt[i] - 1, miss); // miss 不会小于 0，故至多减去 miss 个元素
            }
        }
        return n - miss;
    }
}
```
