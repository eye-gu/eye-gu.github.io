https://leetcode.cn/problems/longest-subarray-with-maximum-bitwise-and/submissions/648252183/



参照2411的方案, 从记录1变成记录0

```java
class Solution {
        public int longestSubarray(int[] nums) {
            // pos[i]代表在nums数组中, 二进制i位上为0的最小下标
            int[] pos = new int[31];
            int ans = 0;
            int max = 0;
            for (int i=nums.length-1; i>=0; i--) {
                int x = nums[i];
                int k = nums.length;
                for (int j=0; j<31; j++) {
                    if ((x & (1 << j)) != 0) {
                        // j位上为1,那么就找这位上为0的最小下标,子数组最长只能到该下标
                        if (pos[j] != 0) {
                            k = Math.min(k, pos[j]);
                        }
                    } else {
                        // j位上为0,那么就记录该下标
                        pos[j] = i;
                    }
                }
                if (x > max) {
                    ans = k - i;
                    max = x;
                } else if (x == max) {
                    ans = Math.max(ans, k - i);
                }
            }
            return ans;
        }
    }
```



结果发现自己傻逼了, 因为是与最大的最长子数组

1. 如果nums[i+1] > nums[i], 那么子数组nums[i+1]的与的结果大于当前与的结果
2. 如果nums[i+1] = nums[i], 加入当前子数组,与的结果不变
3. 如果nums[i+1] < nums[i], 那么把nums[i+1]加进来会导致与的结果变小

说白了就是只能取最大的数字, 最大连续

```go
func longestSubarray(nums []int) int {
	maxVar, cnt, ans := nums[0], 1, 1
	for i := 1; i < len(nums); i++ {
		if nums[i] > maxVar {
			maxVar = nums[i]
			cnt = 1
			ans = 1
		} else if nums[i] == maxVar {
			cnt++
		} else {
			cnt = 0
		}
		ans = max(ans, cnt)
	}
	return ans
}
```
