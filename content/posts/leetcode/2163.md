[2163. 删除元素后和的最小差值 - 力扣（LeetCode）](https://leetcode.cn/problems/minimum-difference-in-sums-after-removal-of-elements/description/)



一开始的想法: 

前1/3剔除a个最大, 可以排序然后计算前缀和
后1/3剔除b个最小, 同理前缀和
a+b <= n
中间部分剩余a+b个, 前a个补充给第一部分, 后b个补充给第二部分, 这里用两个dp数组, 前i个中取j个的最小值, 后i个中取j个最大值

最后先双重循环遍历a和b, 再遍历中间部分的位置, 从dp数组中取 dp1[c] [a]和dp2[n-c] [b]

发现很麻烦, 复杂度也很高



官解

```java
public class Solution {
        public long minimumDifference(int[] nums) {
            int n3 = nums.length, n = n3 / 3;
            long[] part1 = new long[n + 1];
            long sum = 0;
            // 大根堆（用相反数模拟）
            PriorityQueue<Integer> ql = new PriorityQueue<>((a, b) -> b - a);
            for (int i = 0; i < n; ++i) {
                sum += nums[i];
                ql.add(nums[i]);
            }
            // part1[i]代表[0,n+i)的中最小的n个数的和
            part1[0] = sum;
            for (int i = n; i < n * 2; ++i) {
                sum += nums[i];
                ql.add(nums[i]);
                sum -= ql.poll();
                part1[i - (n - 1)] = sum;
            }

            long part2 = 0;
            // 小根堆
            PriorityQueue<Integer> qr = new PriorityQueue<>();
            for (int i = n * 3 - 1; i >= n * 2; --i) {
                part2 += nums[i];
                qr.add(nums[i]);
            }
            long ans = part1[n] - part2;
            for (int i = n * 2 - 1; i >= n; --i) {
                part2 += nums[i];
                qr.add(nums[i]);
                part2 -= qr.poll();
                // 经过上面的计算, 此时part2为[i,n3)中最大的n个数的和
                // part1[i - n]代表[0,i)中最小的n个数的和, 两者相减就是所需的差值
                ans = Math.min(ans, part1[i - n] - part2);
            }
            return ans;
        }
    }
```
