https://leetcode.cn/problems/replace-non-coprime-numbers-in-array/submissions/663274655/



基本是ai生成的, 敲了几个回车, 连一点注释都没有, 只有个方法定义..真是太强了

需要关注的也就是一个gcd计算最大公约数了

```java
class Solution {
    public List<Integer> replaceNonCoprimes(int[] nums) {
        List<Integer> stack = new ArrayList<>();
        for (int num : nums) {
            while (!stack.isEmpty()) {
                int last = stack.get(stack.size() - 1);
                int gcd = gcd(last, num);
                if (gcd == 1) {
                    break;
                }
                stack.remove(stack.size() - 1);
                num = lcm(last, num, gcd);
            }
            stack.add(num);
        }
        return stack;
    }

    private int gcd(int a, int b) {
        if (b == 0) {
            return a;
        }
        return gcd(b, a % b);
    }

    private int lcm(int a, int b, int gcd) {
        return a / gcd * b;
    }
}
```
