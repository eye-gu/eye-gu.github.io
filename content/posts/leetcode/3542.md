https://leetcode.cn/problems/minimum-operations-to-convert-all-elements-to-zero/description/?envType=daily-question&envId=2025-11-14



官解, 单调栈

```java
class Solution {
        public int minOperations(int[] nums) {
            List<Integer> s = new ArrayList<>();
            int res = 0;
            for (int a : nums) {
                while (!s.isEmpty() && s.get(s.size() - 1) > a) {
                    s.remove(s.size() - 1);
                }
                if (a == 0) continue;
                // 如果栈为空或者如果栈顶元素不等于当前元素，说明需要添加一个元素
                if (s.isEmpty() || s.get(s.size() - 1) < a) {
                    res++;
                    s.add(a);
                }
            }
            return res;
        }
    }
```
