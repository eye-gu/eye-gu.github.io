[3202. 找出有效子序列的最大长度 II - 力扣（LeetCode）](https://leetcode.cn/problems/find-the-maximum-length-of-valid-subsequence-ii/description/)



```java
class Solution {
        public int maximumLength(int[] nums, int k) {
            List<Integer>[] remains = new List[k];
            for (int i = 0; i < k; i++) {
                remains[i] = new ArrayList<>();
            }
            // remains[i]存放所有余数等于i的下标
            for (int i = 0; i < nums.length; i++) {
                remains[nums[i] % k].add(i);
            }
            int ans = 0;
            for (int i = 0; i < k; i++) {
                // 所有余数相等的下标, 可以作为一个子序列
                ans = Math.max(ans, remains[i].size());
                for (int j = i + 1; j < k; j++) {
                    // 所有余数两两组合, 计算这种排列的子序列最大长度
                    ans = Math.max(ans, help(remains[i], remains[j]));
                }
            }
            return ans;
        }

        private int help(List<Integer> a, List<Integer> b) {
            if (a.isEmpty() && b.isEmpty()) {
                return 0;
            }
            if (a.isEmpty() || b.isEmpty()) {
                return 1;
            }
            // 待选择的下标
            // 双指针遍历两个下标列表, 都是排好序的, 取比当前下标大的那个
            int i = 0, j = 0;
            // 目前子序列最后一个是a还是b中的下标
            boolean first = true;
            if (a.get(0) > b.get(0)) {
                j++;
                first = false;
            } else {
                i++;
            }
            int ans = 1;
            while (true) {
                if (first) {
                    while (j < b.size() && b.get(j) < a.get(i - 1)) {
                        j++;
                    }
                    if (j == b.size()) {
                        break;
                    }
                    // 子序列选中了j, j本身下移到下一个
                    j++;
                    ans += 1;
                    first = false;
                } else {
                    while (i < a.size() && a.get(i) < b.get(j - 1)) {
                        i++;
                    }
                    if (i == a.size()) {
                        break;
                    }
                    i++;
                    ans += 1;
                    first = true;
                }
            }
            return ans;
        }
    }
```



官解, 这个动态规划的状态定义和转移有点难想啊. 一开始想的都是nums的长度作为dp的一个因素

```java
class Solution {
        public int maximumLength(int[] nums, int k) {
            // dp[i][j] 表示一个有效子序列，最后两个元素模 k 的余数分别是 i 和 j，它的最大长度
            int[][] dp = new int[k][k];
            int res = 0;
            for (int num : nums) {
                // 假设num是子序列的最后一个, 那么子序列最后一个的模就是num %= k
                num %= k;
                for (int prev = 0; prev < k; prev++) {
                    // num和prev间隔的子序列, 且最后一个是num, 那么之前最后两个就是num和prev
                    dp[prev][num] = dp[num][prev] + 1;
                    res = Math.max(res, dp[prev][num]);
                }
            }
            return res;
        }
    }
```
