https://leetcode.cn/problems/the-two-sneaky-numbers-of-digitville/solutions/3804527/shu-zi-xiao-zhen-zhong-de-dao-dan-gui-by-yivz/?envType=daily-question&envId=2025-10-31

官解. 很多位操作的技巧, 最关键的就是异或两次为0. 还有取最后一位1, 然后分组. 然后再[0,n)分组异或, 把真正重复的数找出来.

```java
class Solution {
        public int[] getSneakyNumbers(int[] nums) {
            int n = nums.length - 2;
            int y = 0;
            // 所有不重复数的异或
            for (int x : nums) {
                y ^= x;
            }
            // 再异或[0,n),最终结果就是重复的两个数的异或
            for (int i = 0; i < n; i++) {
                y ^= i;
            }
            // 取最后一位1的小技巧,负数是补码,取反加1
            int lowBit = y & -y;
            int x1 = 0, x2 = 0;
            for (int x : nums) {
                // 根据lowBit分成两组
                if ((x & lowBit) != 0) {
                    x1 ^= x;
                } else {
                    x2 ^= x;
                }
            }
            // 再异或[0,n),x1和x2都少异或了其中一个重复的数
            for (int i = 0; i < n; i++) {
                if ((i & lowBit) != 0) {
                    x1 ^= i;
                } else {
                    x2 ^= i;
                }
            }
            return new int[]{x1, x2};
        }
    }  
```
