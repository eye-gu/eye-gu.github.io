https://leetcode.cn/problems/minimum-number-of-people-to-teach/description/



一开始以为可以选多个语言教, 找最小的教的次数, 感觉也太难了, 原来只能用一门语言.

```java
class Solution {
    public int minimumTeachings(int n, int[][] languages, int[][] friendships) {
        int m = languages.length;
        Set<Integer>[] languagesSet = new Set[m];
        for (int i = 0; i < m; i++) {
            languagesSet[i] = new HashSet<>();
            for (int lang : languages[i]) {
                languagesSet[i].add(lang);
            }
        }
      	// 所有需要学习语言的用户id
        Set<Integer> needLearn = new HashSet<>();
        for (int[] friendship : friendships) {
            int a = friendship[0] - 1;
            int b = friendship[1] - 1;
            boolean need = true;
            for (int l : languages[a]) {
                if (languagesSet[b].contains(l)) {
                    need = false;
                    break;
                }
            }
            if (need) {
                needLearn.add(a);
                needLearn.add(b);
            }
        }
      	// 遍历所有语言,教给需要学习语言的用户, 找最小解
        int res = needLearn.size();
        for (int i = 1; i <= n; i++) {
            int count = 0;
            for (int j : needLearn) {
                if (!languagesSet[j].contains(i)) {
                    count++;
                }
            }
            res = Math.min(res, count);
        }
        return res;
    }
}
```



官解在最后找最小教的次数的时候, 用了另一个思路: 在需要学习语言的用户中,会的人最多的一门语言,就是教的次数最小的语言.

```java
public int minimumTeachings(int n, int[][] languages, int[][] friendships) {
  	// 所有需要学习语言的用户id
    Set<Integer> cncon = new HashSet<>();
    for (int[] friendship : friendships) {
        Map<Integer, Integer> mp = new HashMap<>();
        boolean conm = false;
        for (int lan : languages[friendship[0] - 1]) {
            mp.put(lan, 1);
        }
        for (int lan : languages[friendship[1] - 1]) {
            if (mp.containsKey(lan)) {
                conm = true;
                break;
            }
        }
        if (!conm) {
            cncon.add(friendship[0] - 1);
            cncon.add(friendship[1] - 1);
        }
    }
    int max_cnt = 0;
    int[] cnt = new int[n + 1];
    for (int friendship : cncon) {
        for (int lan : languages[friendship]) {
            cnt[lan]++;
            max_cnt = Math.max(max_cnt, cnt[lan]);
        }
    }
    return cncon.size() - max_cnt;
}
```
