https://leetcode.cn/problems/find-the-k-th-character-in-string-game-ii/description

```java
class Solution {
        public char kthCharacter(long k, int[] operations) {
            // 获取大于等于k的最小的2的n次方
            int i = 63 - Long.numberOfLeadingZeros(k);
            if ((1L << i) < k) {
                i++;
            }
            int ans = 0;
            // 已经增长到k个字符，在i后面的操作都是不需要的
            for (int j = i - 1; j >= 0; j--) {
                // k在前半部分，那么此次操作不影响k
                if ((1L << j) >= k) {
                    continue;
                }
                // k在后半部分
                //  如果k是最后一位，即等于2的n次方，那么减半，即8这个位置上的字符是4位置上的字符递推过来的
                //  如果不是，那么减去2的n次方，即7位置上的字符是3递推过来的
                int n = 63 - Long.numberOfLeadingZeros(k);
                if ((1L << n) == k) {
                    k /= 2;
                } else {
                    k -= (1L << n);
                }

                if (operations[j] == 1) {
                    ans += 1;
                }
            }
            return (char) ('a' + ans % 26);
        }
    }
```

官解，不需要遍历`operations`，从k中获取到当前2进制最高次，就可以对应第几次操作，即第3，4个字符来自于第2次操作，第5，6，7，8个字符来自于第3次操作。

```java
class Solution {
        public char kthCharacter(long k, int[] operations) {
            int ans = 0;
            int t;
            while (k != 1) {
                t = 63 - Long.numberOfLeadingZeros(k);
                if ((1L << t) == k) {
                    t--;
                }
                // k是2的t次方，减半。否则减去2的t次方
                k = k - (1L << t);
                if (operations[t] != 0) {
                    ans++;
                }
            }
            return (char) ('a' + (ans % 26));
        }
    }
```
