[1695. 删除子数组的最大得分 - 力扣（LeetCode）](https://leetcode.cn/problems/maximum-erasure-value/description/)



```java
class Solution {
        public int maximumUniqueSubarray(int[] nums) {
            // preSum[i]代表[0,i)之和
            int[] preSum = new int[nums.length + 1];
            for (int i =0; i< nums.length; i++) {
                preSum[i + 1] = preSum[i] + nums[i];
            }
            int ans = 0;
            // start[i]代表nums[i]结尾符合条件的子数组的起始索引, 即[start[i], i]是一个合法子数组
            int[] start = new int[nums.length];
            for (int i = 0; i < nums.length; i++) {
                int j = i - 1;
                int limit = 0;
                while (j >= limit) {
                    if (nums[j] == nums[i]) {
                        break;
                    }
                    limit = Math.max(limit, start[j]);
                    j--;
                }
                start[i] = j + 1;
                ans = Math.max(ans, preSum[i + 1] - preSum[start[i]]);
            }
            return ans;
        }
    }
```

用前缀和优化子数组和的计算

```java
class Solution {
        public int maximumUniqueSubarray(int[] nums) {
            // preSum[i]代表[0,i)之和
            int[] preSum = new int[nums.length + 1];
            for (int i =0; i< nums.length; i++) {
                preSum[i + 1] = preSum[i] + nums[i];
            }
            int ans = 0;
            int start = 0;
            Map<Integer, Integer> lastIndexMap = new HashMap<>();
            for (int i = 0; i < nums.length; i++) {
                // nums[i]上次出现的下标
                Integer lastIndex = lastIndexMap.get(nums[i]);
                // 如果nums[i]之前就存在, 那么分两种情况
                // 如果lastIndex > start, 那么子数组的开始位置就为lastIndex + 1
                // 如果lastIndex < start, 那么子数组的开始位置不用变
                if (lastIndex != null) {
                    start = Math.max(start, lastIndex + 1);
                }
                ans = Math.max(ans, preSum[i + 1] - preSum[start]);
                lastIndexMap.put(nums[i], i);
            }
            return ans;
        }
    }
```

按照官解的方案, 官解把计算前缀和合并到一次循环了
