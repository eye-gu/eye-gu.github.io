https://leetcode.cn/problems/minimum-score-triangulation-of-polygon/description/?envType=daily-question&envId=2025-09-29



1. 状态定义：dp[i][j] 表示顶点 i 到顶点 j 之间（包括 i 和 j）的多边形区域的最小三角剖分分数。
2. 状态转移：对于区间 [i,j]，我们选择一个中间点 k（i < k < j），连接 i 和 j 形成三角形 (i, k, j)，这样就将原问题分解为两个子问题：
   区间 [i,k] 的最优解
   区间 [k,j] 的最优解
   加上三角形 (i,k,j) 的分数 values[i] * values[k] * values[j]
3. 转移方程   dp[i][j] = min{dp[i][k] + dp[k][j] + values[i] * values[k] * values[j]}  (i < k < j)
4. 边界条件：当 j - i < 2 时，即只有两个或更少的顶点时，无法形成三角形，值为0。
5. 计算顺序：按照区间长度从小到大计算

```java
static class Solution {
    public int minScoreTriangulation(int[] values) {
        int n = values.length;
        // dp[i][j] 表示从顶点i到顶点j的多边形的最小三角剖分分数
        int[][] dp = new int[n][n];

        // 区间DP，区间长度从3开始（最少3个顶点才能组成三角形）
        for (int len = 3; len <= n; len++) {
            for (int i = 0; i <= n - len; i++) {
                int j = i + len - 1;
                dp[i][j] = Integer.MAX_VALUE;
                // 尝试不同的顶点k作为三角形(i, j, k)的第三个顶点
                for (int k = i + 1; k < j; k++) {
                    dp[i][j] = Math.min(dp[i][j], 
                        dp[i][k] + dp[k][j] + values[i] * values[j] * values[k]);
                }
            }
        }

        return dp[0][n-1];
    }
}
```
