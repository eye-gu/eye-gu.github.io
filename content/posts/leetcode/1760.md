https://leetcode.cn/problems/minimum-limit-of-balls-in-a-bag/description/



二分

假设最大的球数为y, 那么nums[i]需要进行`(nums[i] - 1) / y`次操作

全部袋子的操作数累加后总的操作数, 如果小于maxOperations, 那么可以减小y; 如果大于maxOperations,增大y

```go
func minimumSize(nums []int, maxOperations int) int {
	maxNum := 0
	for _, x := range nums {
		if x > maxNum {
			maxNum = x
		}
	}
	return sort.Search(maxNum, func(y int) bool {
		if y == 0 {
			return false
		}
		ops := 0
		for _, x := range nums {
			ops += (x - 1) / y
		}
		return ops <= maxOperations
	})
}
```



`func sort.Search(n int, f func(int) bool) int`

n表示二分查找的范围

f是一个函数, 返回true表示在[0,i]的范围中; false表示在[i+1,n]的范围中
