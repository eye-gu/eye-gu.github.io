https://leetcode.cn/problems/adjacent-increasing-subarrays-detection-ii/description/?envType=daily-question&envId=2025-10-15

```java
class Solution {
        public int maxIncreasingSubarrays(List<Integer> nums) {
            int increasing = 1;
            int preIncreasing = 0;
            int res = 0;
            for (int i = 1; i < nums.size(); i++) {
                if (nums.get(i) > nums.get(i - 1)) {
                    increasing++;
                } else {
                    // 注意一个长的递增数组分成2半
                    res = Math.max(res, Math.max(Math.min(increasing, preIncreasing), increasing / 2));
                    preIncreasing = increasing;
                    increasing = 1;
                }
            }
            res = Math.max(res, Math.max(Math.min(increasing, preIncreasing), increasing / 2));
            return res;
        }
    }
```
