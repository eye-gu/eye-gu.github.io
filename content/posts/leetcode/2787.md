https://leetcode.cn/problems/ways-to-express-an-integer-as-sum-of-powers/description/



使用dfs加记忆化

```java
class Solution {

        public int numberOfWays(int n, int x) {
            Map<Integer, Integer> memo = new HashMap<>();
            return dfs(n, x, n, memo);
        }

        private int dfs(int remain, int x, int max, Map<Integer, Integer> memo) {
            if (max <= 0) {
                return 0;
            }
            int key = remain * 1000 + max;
            if (memo.containsKey(key)) {
                return memo.get(key);
            }
            int count = 0;
            for (int i = Math.min(max, remain); i > 0; i--) {
                long p = 1;
                for (int j = 0; j < x; j++) {
                    p *= i;
                }
                if (p > remain) {
                    continue;
                }
                if (p == remain) {
                    count++;
                    continue;
                }
                count += dfs((int) (remain - p), x, i - 1, memo);
                count %= 1000000007;
            }
            memo.put(key, count);
            return count;
        }
    }
```



官解, dp, 注意的点是从后往前更新，避免重复使用同一个数

```java
class Solution {
        public int numberOfWays(int n, int x) {
            // dp[i] 表示和为 i 的方案数
            int[] dp = new int[n + 1];
            dp[0] = 1; // 和为0的方案数是1（什么都不选）
            
            // 遍历每个可能的底数
            for (int i = 1; ; i++) {
                // 计算 i 的 x 次幂
                long power = 1;
                for (int j = 0; j < x; j++) {
                    power *= i;
                }
                
                // 如果幂值超过 n，就停止
                if (power > n) {
                    break;
                }
                
                int intPower = (int) power;
                // 从后往前更新，避免重复使用同一个数
                for (int j = n; j >= intPower; j--) {
                    dp[j] = (dp[j] + dp[j - intPower]) % 1000000007;
                }
            }
            
            return dp[n];
        }
    }
```
