https://leetcode.cn/problems/maximum-distance-in-arrays/



第一次提交没有考虑最大和最小在同一个数组中

后面尝试记录下第二小和第二大, 还有最大和最小的数组index, 发现相当麻烦, 需要考虑很多

查看官解, 每次遍历新数组的时候, 计算当前的最大距离即可

```go
func maxDistance(arrays [][]int) int {
	minNum := arrays[0][0]
	maxNum := arrays[0][len(arrays[0])-1]
	result := 0
	for i := 1; i < len(arrays); i++ {
		result = max(result, max(maxNum-arrays[i][0], arrays[i][len(arrays[i])-1]-minNum))
		minNum = min(minNum, arrays[i][0])
		maxNum = max(maxNum, arrays[i][len(arrays[i])-1])
	}
	return result
}
```
