https://leetcode.cn/problems/count-subarrays-with-majority-element-i/description/?envType=daily-question&envId=2026-06-25



```java
class Solution {
        public int countMajoritySubarrays(int[] nums, int target) {
          // 前缀和, 记录0到当前为止,target的数量
            int[] preSum = new int[nums.length + 1];
            int pre = 0;
            int count = 0;
            for (int i = 0; i< nums.length; i++) {
                if (nums[i] == target) {
                    pre++;
                }
                preSum[i+1] = pre;

                for (int j=0; j<=i; j++) {
                    if (2 * (preSum[i+1] - preSum[j]) > (i - j + 1)) {
                        count++;
                    }
                }
            }
            return count;
        }
    }
```



题解中O(n)的算法.

因为s是连续的,每次只能+1或者-1,因此f可以只加或者减当前前缀和s出现的次数,就能维护所有小于P[k]的个数.

```java
class Solution {
        public int countMajoritySubarrays(int[] nums, int target) {
            // 权重：target -> +1，其余 -> -1。记 P[k] 为前缀和（P[0]=0）。
            // 子数组 nums[l..r] 中 target 为多数  <=>  P[r+1] > P[l]（0<=l<=r）。
            // 因此对每个 r，统计满足 P[l] < P[r+1] 的 l 的个数。利用前缀和每次只变化 ±1，
            // 可对个数增量维护，无需重新扫描。
            Map<Integer, Integer> cnt = new HashMap<>(); // cnt[v] = 前缀和 v 已出现的次数
            cnt.put(0, 1); // 初始前缀和 P[0]=0,即前缀和0出现 1 次（键=前缀和值，值=出现次数）
            int ans = 0;   // 合法子数组总数
            int s = 0;     // 当前前缀和 P[k]（处理 nums[k] 之前）
            int f = 0;     // 满足 P[l] < P[k] 的 l 个数（l 取值 0..k-1）
            for (int x : nums) {
                if (x == target) {
                    // P[k+1] = P[k] + 1。P[l] < P[k]+1  <=>  P[l] <= P[k]，
                    // 故新满足条件的个数 = f + #{l : P[l] == P[k]}。
                    f += cnt.getOrDefault(s, 0);
                    s++;
                } else {
                    // P[k+1] = P[k] - 1。P[l] < P[k]-1  <=>  P[l] <= P[k]-2，
                    // 故新满足条件的个数 = f - #{l : P[l] == P[k]-1}。
                    s--;
                    f -= cnt.getOrDefault(s, 0);
                }
                // 此时 s == P[k+1]，f = #{l in [0..k] : P[l] < P[k+1]} = 以 k 结尾的合法子数组数。
                ans += f;
                cnt.merge(s, 1, Integer::sum); // 记录 P[k+1]，供后续迭代使用
            }
            return ans;
        }
    }
```
