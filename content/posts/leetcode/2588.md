https://leetcode.cn/problems/count-the-number-of-beautiful-subarrays/description/



这是一开始的想法, 思路是计算每一位上的1的前缀和, 因为美丽数组对于二进制中第 *i* 位，数组中所有元素第 *i* 位为 1 的数目一定为偶数.

最后用一个三层循环(n * n * 32)去判断, 显然因为超时失败了

```go
func beautifulSubarrays(nums []int) int64 {
	// 前缀和, preSum[i][j]代表nums[0..i)所有数字, 第j位的1的个数
	preSum := make([][]int, len(nums)+1)
	for i := 0; i < len(preSum); i++ {
		preSum[i] = make([]int, 31)
	}

	for i := 0; i < len(nums); i++ {
		num := nums[i]

		for j := 0; j < 31; j++ {
			if num&(1<<j) > 0 {
				preSum[i+1][j] = preSum[i][j] + 1
			} else {
				preSum[i+1][j] = preSum[i][j]
			}
		}
	}

	var result int64 = 0

	check := func(i, j int) bool {
		for k := 0; k < 31; k++ {
			if (preSum[j+1][k]-preSum[i][k])&1 == 1 {
				return false
			}
		}
		return true
	}

	for i := 0; i < len(nums); i++ {
		for j := i; j < len(nums); j++ {
			if check(i, j) {
				result++
			}
		}
	}
	return result
}
```



官解

美丽数组因为每一位都是偶数个1, 因此有个很重要的性质是异或为0!!!

因此假如nums[i..j]是美丽数组, 那么nums[i..j]的异或是0, 那么nums[0..i-1]的异或和num[0..j]的异或是相同的

因此计算num[0..j]的异或后, 只要查询前面已经有多少个相同的异或即可

```go
func beautifulSubarrays(nums []int) int64 {
	preOrSum := make(map[int]int)
	// 假如nums[0..i]异或为0, 那么nums[0..i]本身就是一个美丽数组, 所以初始化
	preOrSum[0] = 1
	or := 0
	var result int64 = 0
	for i := 0; i < len(nums); i++ {
		or ^= nums[i]
		n := preOrSum[or]
		result += int64(n)
		preOrSum[or]++
	}
	return result
}
```
