## 神经网络

A0 = X

A1 = sigmoid(W0 * X + b0)

A2 = sigmoid(W1 * a1 + b1)

## 激活函数

### sigmoid

sigmoid = 1/(1+e^-z)

sigmoid(z)的导数是 sigmoid(z) * (1 - sigmoid(z))

### tanh
tanh = (e\^z-e^\-z)/(e\^z+e\^-z), 输出在-1到1之间. 输出均值为0.

tanh(z)的导数是 1 - tanh(z)^2

几乎总是比sigmoid好用. 除了输出层是一个二分类时, sigmoid输出0,1的概率



```python
x = np.linspace(-10,10,100)
y = np.tanh(x)
plt.plot(x,y)
```

### ReLU

relu = max(0,z)

relu(z)的导数是 1 if z >= 0 else 0

目前最常用的激活函数.

### Leaky ReLU

lrelu = max(0.01*z,z), 0.01是一个很小的超参数.

lrelu(z)的导数是 1 if z >= 0 else 0.01


```python
x = np.linspace(-10,10,100)
y = np.maximum(0.01 * x, x)
plt.plot(x,y)
plt.plot(x,np.zeros_like(x), color='red')
```
