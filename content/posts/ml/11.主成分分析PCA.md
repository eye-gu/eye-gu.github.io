无监督学习. 

可以将多个特征压缩到2到3个特征, 进行可视化

1. 特征缩放

2. 选择方差最大的轴, 第二个轴和第一个轴垂直, 第三个轴和前两个轴垂直

x = [2, 3]

轴的单位向量z为 = [0.71, 0.71]

那么x在轴上的的投影为 x * z = 2*0.71 + 3*0.71 = 3.55

### 重构

按照之前的例子, 现在知道投影为3.55, 轴的单位向量z为 = [0.71, 0.71], 如何知道原始的x

事实上, 这是无法得到的, 一个近似的方案是 3.55 * z = [2.52, 2.52]


```python
from sklearn.decomposition import PCA
import numpy as np
import matplotlib.pyplot as plt
```


```python
X = np.array([[1, 1], [2, 1], [3, 2], [-1, -1], [-2, -1], [-3, -2]])
# 省略数据预处理
```


```python
pca_1 = PCA(n_components=1)
pca_1.fit(X)
# 解释了99.2%的原始数据的方差
pca_1.explained_variance_ratio_
```




    array([0.99244289])




```python
X_trans_1 = pca_1.transform(X)
# 重构
X_reduced_1 = pca_1.inverse_transform(X_trans_1)
```


```python
plt.scatter(X[:, 0], X[:, 1], color='blue')
plt.scatter(X_reduced_1[:, 0], X_reduced_1[:, 1], color='red')
plt.plot(X_reduced_1[:, 0], X_reduced_1[:, 1], color='black')
plt.show()

print(X_trans_1)
```
