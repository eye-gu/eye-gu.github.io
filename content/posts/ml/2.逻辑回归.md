## sigmoid函数

g(z) = 1 / (1 + e^(-z))

e = 0 , g(z) = 0.5

e > 0, g(z) > 0.5, 且e越大, 越接近1

e < 0, g(z) < 0.5, 且e越小, 越接近0


```python
def sigmoid(x):
    return 1 / (1 + np.exp(-x))
```

F(w,b)(X) 的取值范围是(0, 1)

## 决策边界

## 成本函数

在线性回归中, 损失函数是 `(F(w,b)(X) - y)^2`, 在逻辑回归中并不适用.

因此首先定义损失函数loss如下:

if y = 1:

    - log(F(w,b)(X))

if y = 0:

    - log(1 - F(w,b)(X))

查看以下两张图

在y=1时, F(w,b)(X)越接近1, 那么成本越小, 如果恰好等于1那么成本就是0. F(w,b)(X)越接近0, 成本越大, 且指数级增长. 

在y=0时, F(w,b)(X)越接近0, 那么成本越小, 如果恰好等于0那么成本就是0. F(w,b)(X)越接近1, 成本越大, 且指数级增长.

最终的成本函数就是sum(loss) / m

### 损失函数的简易表达

`-ylog(F(w,b)(X)) - (1-y)log(1 - F(w,b)(X))`

在y等于0或者等于1的时候, 该式左右只有一边有值

成本函数就可以表示为: `-sum(ylog(F(w,b)(X)) + (1-y)log(1 - F(w,b)(X))) / m`

这个成本函数是通过统计学的最大似然估计得到的, 它是凸的


```python
def cost(X, y, w, b):
    z = np.dot(X, w) + b
    g = sigmoid(z)
    return -np.sum(y*np.log(g) + (1-y)*np.log(1-g))
```

## 梯度下降

### 梯度

首先`log(x)` 求导的结果是 `1/x`, `e^x` 求导为 `e^x`.

`loss = -ylog(1 / (1 + e^(- w * X - b))) - (1-y)log(1 - 1 / (1 + e^(- w * X - b)))`

`g(z) = 1 / (1 + e^(- w * X - b))`

因此 `loss = -ylog(g(z)) - (1-y)log(1-g(z))`

先求loss对g(z)求导:
1. `-y / g(z) + (1-y) / (1-g(z))` 中间的减号变成加号, 是因为`-g(z)`求导为-1
2. 两边通分 `(-y(1 - g(z)) + g(z)(1 - y)) / g(z) / (1 - g(z))`
3. `(g(z) - y) / g(z) / (1 - g(z))`


再求g(z)对z求导, `z=(w * X + b)`, `g(z) = 1 / (1 + e^-z)`:
1. e^-z / (1 + e^-z)^2
2. 变换为 1 / (1 + e^-z) * (1 - 1 / (1 + e^-z))
3. 就等于 g(z) * (1 - g(z))


将两个结果相乘:
1. `(g(z) - y) / g(z) / (1 - g(z)) * g(z) * (1 - g(z))`
2. 约去后等于 `g(z) - y`

最后是z对w求导, 结果就是X. 

因此最终的结果就是 `(g(z) - y)X`, 展开就是 `(1 / (1 + e^(- w * X - b)) - y) * X`

### 梯度下降

`w = w - alpha * (g(z) - y) * X / m`

`b = b - aplha * (g(z) - y) / m`



```python
def compute_gradient(X, y, w, b):
    z = np.dot(X, w) + b
    g = sigmoid(z)
    dw = np.dot(X.T, (g - y)) / y.size
    db = np.sum(g - y) / y.size
    return dw, db
```


```python
import copy

def gradient_descent(X, y, w, b, learning_rate = 0.01, num_iterations=1000):
    w = copy.deepcopy(w)
    cost_history = []
    for i in range(num_iterations):
        dw, db = compute_gradient(X, y, w, b)
        w -= learning_rate * dw
        b -= learning_rate * db
        cost_history.append(cost(X, y, w, b))
    return w, b, cost_history
```


```python
X_tmp = np.array([[0.5, 1.5], [1,1], [1.5, 0.5], [3, 0.5], [2, 2], [1, 2.5]])
y_tmp = np.array([0, 0, 0, 1, 1, 1])
w_tmp  = np.zeros_like(X_tmp[0])
b_tmp  = 0.
alph = 0.0001
iters = 10000

w_out, b_out, cost_history = gradient_descent(X_tmp, y_tmp, w_tmp, b_tmp, learning_rate = alph, num_iterations= iters) 
print(f"\nupdated parameters: w:{w_out}, b:{b_out}")

plt.plot(np.arange(iters), cost_history, label='cost', c='blue')
```
