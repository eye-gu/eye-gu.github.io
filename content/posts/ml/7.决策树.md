## 熵

p1为样本中正样本的比例, 即正样本的数量/所有样本数量. 

p0 = 1-p1

熵定义: `H(p) = - p1 * log2(p1) - p0 * log2(p0) = - p1 * log2(p1) - (1-p1) * log2(1-p1)`


```python
import numpy as np
import matplotlib.pyplot as plt
```


```python
def h(p1):
    epsilon = 1e-10
    p1 = np.clip(p1, epsilon, 1 - epsilon)
    return -p1 * np.log2(p1) - (1 - p1) * np.log2(1 - p1)
```


```python
x = np.arange(0, 1.01, 0.01)
y = h(x)
plt.plot(x, y)
plt.xlabel('p1')
plt.ylabel('H')
plt.show()
```

### 纯度

### 信息增益

p1_left: 左分支的正样本的比例

w_left: 左分支占根节点的样本比例, num_left / num_root

p1_right: 右分支的正样本的比例

w_right: 右分支占根节点的样本比例, num_right / num_root

p1_root: 根节点的正样本的比例

信息增益 = H(p1_root) - (w_left * H(p1_left) + w_right * H(p1_right))

## 停止标准

1. 节点全部都是同个类型
2. 超过最大深度
3. 信息增益小于阈值
4. 节点样本数量小于阈值

## 连续值

通过一个阈值对数据切分, 阈值的选择还是取最大的信息增益.

对根节点的数据排序, 可能的阈值选择有n-1种.

## 回归树

### 如何进行预测

假设已经有一颗分割好的决策树, 那么最后的预测值为叶子节点所有样本数量的平均值

### 如何分割

不再计算加权熵, 而是加权方差

w_left: 左分支占根节点的样本比例, num_left / num_root

w_right: 右分支占根节点的样本比例, num_right / num_root

`根节点方差 - w_left * 左方差 + w_right * 右方差`

## 树集成

构造多个决策树, 然后对结果进行投票

### 有放回抽样

### 随机森林

通过有放回抽样构造多个随机的样本集, 在每个样本集上训练一颗决策树.

一般在64到128颗树

### 随机特征选择

在构造决策树时, 不从所有特征中进行选择, 而是选择k个子集的特征中, 选择信息增益最大的特征. 一般k的取值是平方根的总特征数

### xgboost

在生成下一颗树的有放回抽样时, 目前生成的所有树, 在原始样本集上分类错误的样本将更有可能被抽样到.
