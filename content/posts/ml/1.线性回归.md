### 梯度下降

梯度就是成本函数, 分别对w,和b求导.

最外层的平方求导就是乘2: 2*(np.sum(X * w + b - Y)) / 2 / m

2约去: (np.sum(w * X + b - Y)) / m

内部对w求导的话就是乘x: X * (np.sum(w * X + b - Y)) / m

内部对b求导就是1: (np.sum(w * X + b - Y)) / m




```python
def compute_gradient(X, Y, w, b):
    predict = X * w + b
    delta = predict - Y
    dw = np.sum(delta * X) / X.shape[0]
    db = np.sum(delta) / X.shape[0]
    return dw, db
```


```python
def gradient_descent(X, Y, w, b, alpha=0.01, num_iters= 10000):
    J_history = []
    p_history = []
    for i in range(num_iters):
        dw, db = compute_gradient(X, Y, w, b)
        w -= alpha * dw
        b -= alpha * db
        cost = compute_cost(X, Y, w, b)
        J_history.append(cost)
        p_history.append((w, b))
    return w, b, J_history, p_history
```

### 梯度下降

f = w1 * x1 + w2 * x2 + w3 * x3 + b

cost = (f - y) ** 2 / 2 / m

所以:

对w1求导: (f - y) * x1 / m

对w2求导: (f - y) * x2 / m

...

对b求导: (f - y) / m


```python
def compute_gradient(X, y, w, b):
    m, n = X.shape
    dw = np.zeros(n)
    db = 0

    for i in range(m):
        predict = X[i].dot(w) + b
        # delta是个标量
        delta = predict - y[i]
        dw += delta * X[i]
        db += delta
    return dw / m, db / m
```

```python
def gradient_descent(X, y, w, b, compute_cost, compute_gradient, alpha=0.01, num_iters=10000):
    w = copy.deepcopy(w)
    J_history = []
    for _ in range(num_iters):
        dw, db = compute_gradient(X, y, w, b)
        w -= alpha * dw
        b -= alpha * db
        cost = compute_cost(X, y, w, b)
        J_history.append(cost)
    return w, b, J_history
```


