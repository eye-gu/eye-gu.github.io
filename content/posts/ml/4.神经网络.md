## 激活函数

目前用的激活函数都是`Sigmoid`. 神经网络更常用的激活函数`ReLU`.

还有一种激活函数为线性激活函数, 就是 g(z) = z

### ReLU

g(z) = max(0, z)


```python
x = np.arange(-10, 10, 0.1)
y = np.maximum(0, x)
plt.plot(x, y)
plt.axvline(x=0, color='r', linestyle='--')
plt.show()
```

### 激活函数的选择

在为输出层选择激活函数时, 通常根据结果的数据类型,可以自然选择需要的激活函数. 输出层如果是二分, 那么选择sigmoid. 如果是线性回归, 那么选择线性激活. 如果结果是非负的, 例如房价, 那么选择ReLu

隐藏层通常ReLU是最常用的选择, 相较于sigmoid学习得更快. 注意隐藏层不要使用线性激活函数.

## 多分类

在多分类下, 使用`softmax`对所有可能的输出计算概率

`ai = e^zi / (e^z1 + e^z2 + ... + e^zi + ... + e^zn)`

### 代价函数

```
loss = -log(a1) if y=1
     = -log(a2) if y=2
     .
     .
     .
     = -log(an) if y=n
```

在y=i时, 代价函数为 -log(ai), ai的取值范围是0-1, 查看下面的-log图像, 越接近1, 代价越小, 越接近0, 代价越大.


```python
# -logx图像
x = np.arange(0.1, 1, 0.01)
y = - np.log(x)
plt.plot(x, y)
plt.show()
```

```python
# 最后一层从sigmoid改为linear, 损失函数from_logits=True, 将让tensorflow计算更准确, 减小数据损失误差
model = Sequential(
    [
        Dense(units=25, activation='relu'),
        Dense(units=15, activation='relu'),
        Dense(units=10, activation='linear')
    ]
)

model.compile(
    loss=tf.keras.losses.BinaryCrossentropy(from_logits=True)
)
```


```python
model = Sequential(
    [
        Dense(units=25, activation='relu'),
        Dense(units=15, activation='relu'),
        Dense(units=10, activation='softmax')
    ]
)

# SparseCategoricalCrossentropy 稀疏分类交叉熵函数
# 稀疏分类: 仍旧是分类, 但只能是结果中的一种, 不能同时是a又是b
model.compile(
    loss=tf.keras.losses.SparseCategoricalCrossentropy()
)
```


```python
# 同样的, 减小数据损失误差
model = Sequential(
    [
        Dense(units=25, activation='relu'),
        Dense(units=15, activation='relu'),
        Dense(units=10, activation='linear')
    ]
)

model.compile(
    loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True)
```

## Adam

自动调整学习率. 相较于原始的梯度下降, 学习得更快.


```python
model = Sequential(
    [
        Dense(units=25, activation='relu'),
        Dense(units=15, activation='relu'),
        Dense(units=10, activation='linear')
    ]
)

model.compile(
    loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
    optimizer=tf.keras.optimizers.Adam(learning_rate=1e-3)
)
```
